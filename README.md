# Serene-Daily
## Overview
Serene Daily is a simple mental health journaling app that allows users to create written entries, and is designed to help those who are struggling with any kind of mental health issue. This app allows users to write down any and all emotions and thoughts, negative and positive, to help them understand their emotions better. 

Created for my Capstone Project class. 

## Features
* __Journal Entries:__ Users can create as many entries as they wish.
* __Read Entry:__ Users can choose a specific entry and have it written out.
* __Edit Entries:__ Users can choose any entry to add content.
* __View Entries:__ Users can look back at all older entries.
