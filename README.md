# Serene-Daily
## Overview
Serene Daily is a simple mental health journaling app that allows users to create written entries, and is designed to help those who are struggling with any kind of mental health issue. This app allows users to wrtie down any and all emotions and thoughts, negative and positive, to help them understand their emotions better. 

## Features
  <li>Journal Entries: Users can create as many entries as they wish.</li>
  <li>Easy to use interface</li>
  <li>Journal: Users can look back at any older entries.</li>
